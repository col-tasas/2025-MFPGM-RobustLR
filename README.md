# 2025-MFPGM-RobustLR
This repository contains code from our paper "Sample-Efficient Model-Free Policy Gradient Methods for Stochastic LQR: via Robust Linear Regression"

1. Run MainCode.m and then you can obtain file Data.mat.
2. Run NPGwithfixedstepsize.m and then you can obtain file NPG.mat
3. Run FigureGeneration and then you can generate the plot.
